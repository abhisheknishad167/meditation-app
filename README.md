# 🧘 Meditation Timer App

A minimal and calming meditation timer built using **Jetpack Compose** and modern Android architecture.  
Designed to provide a **distraction-free experience** with smooth animations and a clean user interface.

It helps users stay focused during meditation sessions with a simple timer, breathing visuals, and subtle sound feedback.

---

## ✨ Key Highlights
- Modern UI built with **Jetpack Compose**
- Smooth breathing animation for guided relaxation
- Customizable meditation duration
- Minimal and distraction-free design
- Lightweight and performance-friendly

---

## 🛠 Tech Stack
- **Kotlin**
- **Jetpack Compose**
- **MVVM Architecture**
- **StateFlow** for state management

---

## 🎯 Goal
To create a simple, elegant, and effective meditation experience that helps users relax and stay mindful without unnecessary complexity.
